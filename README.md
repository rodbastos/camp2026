# CAMP 2026

Página de divulgação do CAMP 2026 — uma desconferência para quem cultiva sistemas sociais mais vivos.

- **Data:** 15 a 18 de outubro de 2026
- **Local:** Vila dos Portões, Araçariguama - SP
- **Inscrições:** https://forms.gle/Gq7Hmm1qiVkSoU2p6

## Stack

- [Next.js 14](https://nextjs.org/) (App Router)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide Icons](https://lucide.dev/)
- Deploy: [Netlify](https://www.netlify.com/)

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse http://localhost:3000.

## Imagem do hero

Adicione uma foto em `public/hero.jpg` e substitua o placeholder no componente hero em `app/page.tsx`.
